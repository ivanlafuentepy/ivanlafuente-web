Paso 1 · La herramienta

Claude Code, dentro de
la app Claude Desktop

Claude Code es la versión de Claude que trabaja en tu compu: crea archivos, programa y enciende tu agente. Y vive adentro de la app Claude Desktop que ya conocés — sin terminal aparte, sin instalar nada raro.

La misma cuenta: es el mismo Claude del Nivel 1. Le hablás en español y él hace el trabajo técnico por vos.

Paso 3 · Hacelo tuyo

Cargale los datos de tu negocio

El agente base es genérico. Se vuelve tuyo cuando le cargás quién sos: nombre, qué vendés, precios, horarios y el tono con el que querés que hable.

El mismo agente, cualquier negocio: cambiás estos datos y la misma base sirve para una academia, un consultorio o una tienda.
